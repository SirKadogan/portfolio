# Retro Portfolio

A developer portfolio with a unique 8-bit aesthetic that combines nostalgia with modern web development practices. Built using Astro's content collections with MDX for enhanced content authoring.

## Features

- **Content Collections**: Organized project showcase using Astro's content system
- **MDX Support**: Rich content authoring with React components
- **Retro Styling**: Authentic 8-bit pixel art design
- **SEO Optimized**: Built-in meta tags and structured data
- **Performance First**: Static site generation for optimal speed

## Tech Stack

- **Framework**: Astro
- **Content**: MDX with content collections
- **Styling**: Tailwind CSS with custom retro theme
- **Language**: TypeScript
- **Deployment**: Vercel

This portfolio showcases how to build a modern, performant website while maintaining a unique and memorable design aesthetic that stands out in today's web landscape.
