# Data Analytics

Developed a comprehensive data analytics solution that consolidates sales data from multiple sources into a single source of truth for accounting and business intelligence.

## Overview

Built an enterprise data pipeline that processes and transforms sales data from various sources including NetSuite, providing unified analytics and reporting capabilities for accounting and business operations.

## Key Features

- **Data Integration**: Multiple source system connectivity
- **ETL Pipeline**: DBT for data transformation
- **Cloud Warehousing**: Snowflake and Redshift integration
- **Single Source of Truth**: Unified data model for accounting
- **Business Intelligence**: Comprehensive reporting and analytics
- **NetSuite Integration**: ERP system data processing

## Tech Stack

- **Data Warehouse**: Snowflake, Redshift
- **ETL**: DBT (Data Build Tool)
- **ERP**: NetSuite
- **Analytics**: Business intelligence and reporting
- **Company**: Whitespectre
