# Warehouse Management System

Developed a comprehensive warehouse management solution that tracks material quantities and locations across global warehouse networks.

## Overview

Built a full-stack application that provides real-time visibility into warehouse operations worldwide. The system enables efficient inventory management and location tracking for complex supply chain operations.

## Key Features

- **Global Operations**: Multi-warehouse support across continents
- **Real-time Tracking**: Live inventory and location updates
- **GraphQL API**: Efficient data fetching with DataLoader optimization
- **Full-stack Development**: Next.js frontend with Node.js backend
- **Scalable Architecture**: Designed for enterprise-level operations

## Tech Stack

- **Frontend**: Next.js
- **Backend**: Node.js
- **API**: GraphQL
- **Performance**: DataLoader
- **Company**: Whitespectre
