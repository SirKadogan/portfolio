# Asset Manager

A comprehensive asset management system that allows customers to modify multiple brand's product detail pages from a centralized dashboard.

## Overview

Built a sophisticated asset management platform that provides customers with granular control over their product detail pages across multiple brands. The system features an intuitive interface with real-time updates and advanced customization options.

## Key Features

- **Multi-brand Support**: Manage multiple brands from one dashboard
- **Real-time Updates**: SWR for efficient data fetching
- **Smooth Animations**: React Spring for fluid interactions
- **Modern UI**: Chakra UI component library
- **Type Safety**: Full TypeScript implementation
- **Performance**: Next.js optimization

## Tech Stack

- **Framework**: Next.js
- **Language**: TypeScript
- **UI Library**: Chakra UI
- **Animations**: React Spring
- **Data Fetching**: SWR
- **Company**: Whitespectre
