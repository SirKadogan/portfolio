# Project Time Tracker

A comprehensive time tracking application designed for enterprise clients including Amazon and EY to monitor and control time spent on various projects.

## Overview

Developed a robust time tracking solution that helps organizations manage project time allocation and productivity. The application provides detailed analytics and reporting capabilities for project management.

## Key Features

- **Multi-platform**: Web and mobile applications
- **Enterprise Integration**: Used by major corporations
- **Real-time Sync**: Cross-device synchronization
- **Analytics Dashboard**: Comprehensive reporting tools
- **Offline Support**: Local database with RealmDB

## Tech Stack

- **Frontend**: React, TypeScript
- **Mobile**: React Native
- **Database**: RealmDB
- **Company**: Apponte.me
