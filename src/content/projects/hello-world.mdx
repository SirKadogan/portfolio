# Hello World App

A retro-styled demo application showcasing modern web development with a nostalgic 8-bit aesthetic. Built with Astro for optimal performance and developer experience.

## Features

- **8-bit Retro Design**: Pixel-perfect styling with authentic retro colors
- **Responsive Layout**: Works seamlessly across all devices
- **Dark Mode Toggle**: Classic light/dark theme switching
- **Performance Optimized**: Built with Astro for lightning-fast loading

## Tech Stack

- **Framework**: Astro
- **Styling**: Tailwind CSS
- **Language**: TypeScript
- **Deployment**: Vercel

This project demonstrates how modern web technologies can be used to create authentic retro experiences while maintaining excellent performance and accessibility standards.
