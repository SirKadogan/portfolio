# Sprint Analytics

Developed an AI-powered analytics system that analyzes Jira sprint data and GitHub comments to identify friction points and provide actionable insights for project management.

## Overview

Built an intelligent analytics platform that processes development workflow data to generate comprehensive sprint reports. The system helps identify bottlenecks and provides management with data-driven insights for process improvement.

## Key Features

- **Data Integration**: Jira and GitHub API integration
- **AI Analysis**: Machine learning models for pattern recognition
- **Friction Detection**: Automated identification of workflow bottlenecks
- **Management Reports**: Actionable insights for decision making
- **Real-time Processing**: Continuous data analysis and reporting

## Tech Stack

- **Backend**: Node.js
- **AI/ML**: Custom AI models
- **APIs**: Jira, GitHub
- **Analytics**: Data processing and visualization
- **Company**: Whitespectre
